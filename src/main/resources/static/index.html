🚀 Project Analyzer

Analyze GitHub, GitLab, and Bitbucket repositories and get interview-ready insights instantly.

📌 General Requirements

  • The repository must be public.
  • A README.md must be present in the root directory.
  • The default branch should be main or master.
  • The README should include:
    • Project overview
    • Tech stack
    • Architecture
    • Features

🌐 Platform Instructions

🐙 GitHub

  • https://github.com/username/repository
  • README in root
  • Branch: main or master

🦊 GitLab

  • https://gitlab.com/username/repository
  • Repository must be public
  • README in root
  • Branch: main or master

🪣 Bitbucket

  • https://bitbucket.org/workspace/repository
  • Do NOT include /src/branch/
  • README in root
  • Branch: main or master
