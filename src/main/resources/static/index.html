📌 General Guidelines

  • Make sure the repository is public.
  • A README.md should be available in the root directory.
  • The default branch is usually main or master.
  • For best results, include:
    • Project overview
    • Tech stack
    • Architecture
    • Features

🌐 Platform Tips

🐙 GitHub

  • https://github.com/username/repository
  • README in root
  • Branch: main or master

🦊 GitLab

  • https://gitlab.com/username/repository
  • Repository should be public
  • README in root
  • Branch: main or master

🪣 Bitbucket

  • https://bitbucket.org/workspace/repository
  • Avoid /src/branch
  • README in root
  • Branch: main or master
